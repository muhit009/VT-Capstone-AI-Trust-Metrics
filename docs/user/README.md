# Boeing Aircraft Assistant — User Documentation

## Overview

The Boeing Aircraft Assistant is a question-and-answer workspace designed to help new hires learn aircraft concepts, comparisons, and engineering terminology. The application provides grounded answers and pairs them with confidence information, source evidence, and a human-readable explanation of why a response should or should not be trusted.

This documentation is intended for:
- first-time users
- demo evaluators
- instructors
- non-developer teammates
- Boeing new-hire style end users

## What the application provides

The application includes:
- a chat-based question interface
- a conversation history sidebar
- a settings page
- a right-side review panel for confidence, signals, evidence, explanation, and metadata

The interface is designed around asking grounded airplane questions and helping the user decide whether the resulting answer is safe to trust. 

## Intended use

Use this system when you want to:
- compare aircraft families (ex. the 737 MAX and 787)
- understand engineering concepts and terms
- inspect citations before reusing an answer
- evaluate whether an answer is trustworthy enough to treat as a draft, reference, or starting point

## Important notes

This tool is intended to support learning and analysis. Users should still verify critical claims, especially when confidence is not high or when evidence is limited. The application’s own confidence guide defines:
- **High** as grounded enough to reuse, while still verifying critical specifications
- **Medium** as useful draft information that still requires checking assumptions and evidence
- **Low** as unsafe until confirmed from source documents
